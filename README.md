# FACILE-IA

Site web de FACILE-IA - Digitalisation PME avec 20 ans d'expertise Airbus appliqués à l'IA

## Installation

```bash
cd facile-ia
npm install
```

## Développement

```bash
cd facile-ia
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
cd facile-ia
npm run build
npm start
```

## Déploiement

Le projet est déployé sur [Vercel](https://vercel.com).
